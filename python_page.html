Python is a programming language that has been growing in popularity in the last few years. Why is it special?

Is it good to start with?

For someone trying to learn how to code for the first time, Python is a great choice. Python's relatively easy learning curve makes it a great language to start off with. The code is written in a way that makes it sound like normal english. It also doesn't have the same amount of boilerplate code, which makes it very easy to just jump in and write.

Disclaimer: Python can be a double edged sword. Even though it's easy to write in Python, switching to a more traditional language may be difficult since they tend to be more complicated.

Is it too slow?

Many people like to write off Python as a slow and useless language. However, it's speed has never been its selling point. Most people can't tell the difference between the bootup speed of programs written in this language versus other faster languages. Companies like for their developers to code quickly and productively, and Python's ease of use allows that.
