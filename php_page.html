I guess I should preface this by saying that PHP is a coding language. Not a good one. Not a fine one. Just a language. However, it is very popular. So popular, in fact, that it is used by more than 75% of all websites, and it's the fourth most popular language. But don't be fooled.

Don't make PHP your first language.

Here's a very basic example of why PHP is a terribly languge. The way PHP handles booleans and truth values is very wierd. For example, the string "foobar" is counted as true. This pretty wierd, but it isn't unheard of for strings to be default true in other languages. However, "foobar" is also equal to 0 for whatever reason. However, 0 isn't equal to true.

TL;DR

In PHP, true == "foobar" == 0, but true != 0.
This should be enough to scare you away. Just use one of the other languages.
